# Book-store-project-in-sql
In this project, SQL queries are used to design, store, manage, and analyze data efficiently. The database is created to maintain structured records, and queries are written to perform operations like data insertion, retrieval, updating, deletion, and analysis.

Operations Performed Using SQL

Data insertion using INSERT INTO

Data retrieval using SELECT

Filtering using WHERE

Sorting using ORDER BY

Grouping using GROUP BY

Aggregate functions like COUNT(), SUM(), AVG()

Tools Used

MySQL Workbench

SQL (DDL & DML Queries)
